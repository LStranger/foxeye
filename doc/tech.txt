constants scheme is x_xxx, used in core are:
A_xxx	client/channel modes
B_xxx	bindtable types
D_xxx	session states
E_xxx	socket errors
F_xxx	requests:logging flags
I_xxx	interface types
M_xxx	socket modes
O_xxx	runtime options
S_xxx	signal types
T_xxx	UI packet types
U_xxx	clientrecord types
W_xxx	wtmp events

debug levels (direct sessions may has debug log only if level<3):
1	data changes
2	search results
3	any other functions calls
4	any transfers but debug itself

listfile (<name> means it can be get via GetField(x,"name")):
record	lname:<passwd>:uid:flags:<charset>:<info>:<.login>:<.logout>:createtime
host	+nick!ident@host
special	 speciallname:flags:expiretime text
other	 name value

speciallnames records:
&#+!	channels	<passwd>=+k <info>=chanmodelock
@	networks	<passwd>=ident <info>=realname <.logout>=network
	bots		<.logout>=network(i.e."botnet","eggbot",etc.)
